在用python重写算法，学习学习算法时，一时好奇（打基础）来学习了python的下划线

  • 单前导下划线：_var 小小警示，提示只在内部使用，非public和private那种

  • 单末尾下划线：var_ 解决命名冲突的问题

  • 双前导下划线：__var 防止变量在子类中被重写，只能在类内访问

  • 单下划线：_ 使用“_”来表示它只是一个临时值

  • 双前导和末尾下划线：

    __var__
